怎用用idea 生成uml图？
https://blog.csdn.net/zj420964597/article/details/87856758

1。右键 包抽象工厂->Diagrams->show Diagram->java class diagram
2。切换左上角的按钮有不同的表现形式
3。打开内层的包，展开为uml 图。
   右键 package 机械工厂->expand nodes。
4. 右键uml途中元素 jump to resource 则 可以查看源码。
5. 右键uml途中元素 layout 更改布局。
5. 右键uml途中元素 show dependencies 查看依赖关系。
