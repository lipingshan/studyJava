    主要作用
    通过反射可以使程序代码访问装载到JVM 中的类的内部信息。
    优点；灵活  用于框架等比如spring的xml<bean></bean>,读取配置文件，jdbc链接等。
    缺点：性能差，破环封装，不便于维护。
